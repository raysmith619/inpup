# Introduction to Programming Using Python
## Setup
* pip install from PyPI
   ```
  python3 -m pip install intro-prog
  ``` 
### Simple Testing / Viewing
Terminal Commands:
```
        ip, iphelp - List intro programming commands
        ipfiles - copy files to Desktop/ip_files
        ipreadme - Display README.md (as seen on PyPI entry)
        iptest - Do simple program test
        iptest2 - simple program test 2

```

### Simple Startup tests
```
iptest
```
or
```
iptest2
```
